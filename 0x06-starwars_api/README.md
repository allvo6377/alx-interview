```Algorithm``` | ```API``` | ```JavaScript```
## Requirements
### General
* Allowed editors: vi, vim, emacs
* All the files will be interpreted on Ubuntu 14.04 LTS using node (version 10.14.x)
* All the files should end with a new line
* The first line of all the files should be exactly #!/usr/bin/node
* A README.md file, at the root of the folder of the project, is mandatory
* Code should be semistandard compliant. Rules of Standard + semicolons on top. Also as reference: [AirBnB style](https://github.com/airbnb/javascript)
* All files must be executable
* The length of the files will be tested using wc
* Not allowed to use var
